## 🔄 Self-Evolving & Agents

### 1. PRECEPT: Planning Resilience via Experience, Context Engineering & Probing Trajectories A Unified Framework for Test-Time Adaptation with Compositional Rule Learning and Pareto-Guided Prompt Evolution

**저자**: Arash Shahmansoori | [원문](https://arxiv.org/abs/2603.09641v1) | [PDF](https://arxiv.org/pdf/2603.09641v1) | **유망점수**: 12점

# PRECEPT 논문 리뷰

## 핵심 기여
기존 LLM 에이전트들은 자연어 기반 지식 저장 시 조건 수가 증가하면 검색 성능이 급격히 저하되고, 학습된 규칙의 신뢰성 있는 조합이 어려우며, 오래된/적대적 지식 탐지 메커니즘이 부재했다. PRECEPT는 (1) 구조화된 조건 키 기반 결정론적 정확 매칭 검색, (2) 베이지안 소스 신뢰도와 임계값 기반 규칙 무효화를 갖춘 충돌 인식 메모리, (3) COMPASS라는 파레토 기반 프롬프트 진화 루프를 통합하여 이 세 문제를 동시에 해결한다.

## 방법론
세 가지 컴포넌트가 긴밀하게 결합된 구조다. 정확 매칭 검색은 부분 매칭 해석 오류를 원천 차단(N=10에서 94.4%→0%)하고, 의미적 계층 구조를 통해 규칙의 조합적 스태킹을 지원한다. 충돌 인식 메모리는 정적-동적 지식 간 불일치를 해소하고 드리프트 적응을 가능케 한다. COMPASS는 동일한 종단간 실행 파이프라인에서 프롬프트를 평가하며 파레토 최적화를 수행한다.

## 실험 결과
9-10 시드 기준으로 Full Reflexion 대비 첫 시도 성공률 +41.1pp(d>1.9), 조합적 일반화 +33.3pp(d=1.55), 2-way 물류 조합에서 100% P1 달성(d=2.64)을 기록했다. 연속 학습에서 +40-55pp 향상, 적대적 정적 지식 하에서 물류 100% 강건성, 드리프트 회복 +55.0pp(p=0.031), 스텝 수 61% 감소를 보였으며 대부분 p<0.001로 통계적 유의성이 확보되었다.

## 한계 및 향후 연구
정확 매칭 검색의 결정론적 특성은 유연성을 제한할 수 있으며, 구조화된 조건 키 설계가 도메인별로 추가 엔지니어링을 요구할 수 있다. 적대적 통합 시나리오에서는 완전 회복이 아닌 부분 회복만 달성되었다. 향후 더 복잡한 다중 에이전트 환경이나 실시간 스트리밍 시나리오로의 확장 연구가 기대된다.

### 2. Test-Driven AI Agent Definition (TDAD): Compiling Tool-Using Agents from Behavioral Specifications

**저자**: Tzafrir Rehan | [원문](https://arxiv.org/abs/2603.08806v1) | [PDF](https://arxiv.org/pdf/2603.08806v1) | **유망점수**: 12점

# Test-Driven AI Agent Definition (TDAD) 논문 리뷰

## 핵심 기여
기존 에이전트 개발은 프롬프트를 수동으로 작성하고 배포 후에야 문제를 발견하는 방식이었다. TDAD는 **프롬프트를 "컴파일된 산출물"로 취급**하여, 행동 명세(spec)로부터 테스트를 자동 생성하고, 테스트가 통과할 때까지 프롬프트를 반복 개선하는 패러다임을 제시한다. 특히 specification gaming 방지를 위해 hidden test, 의미론적 뮤테이션 테스트, 회귀 안전성 측정이라는 세 가지 메커니즘을 도입했다.

## 방법론
엔지니어가 원하는 에이전트 행동을 명세로 작성하면, **첫 번째 코딩 에이전트가 이를 실행 가능한 테스트로 변환**한다. 이후 **두 번째 코딩 에이전트가 테스트를 통과할 때까지 프롬프트를 반복 수정**한다. 핵심 아이디어는 소프트웨어 공학의 TDD(Test-Driven Development)를 LLM 에이전트 프롬프트 엔지니어링에 적용한 것이다. Hidden/visible 테스트 분리로 과적합을 방지하고, 뮤테이션 테스트로 테스트 스위트의 품질을 검증한다.

## 실험 결과
SpecSuite-Core 벤치마크(정책 준수, 분석, 런북 준수, 결정적 강제 등 4가지 에이전트)에서 24회 독립 실험 결과, **v1 컴파일 성공률 92%, hidden 테스트 평균 통과율 97%**를 달성했다. 진화된 명세(v2)에서는 컴파일 성공률 58%, hidden 통과율 78%, **뮤테이션 점수 86-100%**, 회귀 안전성 97%를 기록했다. 이는 TDAD가 명세 변경 시에도 안정적으로 에이전트를 유지할 수 있음을 보여준다.

## 한계 및 향후 연구
벤치마크가 4개의 특정 도메인 에이전트로 제한되어 일반화 검증이 부족하다. 또한 v2 명세에서 컴파일 성공률이 58%로 하락하여 복잡한 명세 진화에는 개선이 필요하다. 향후 더 다양한 도메인, 더 복잡한 다중 에이전트 시스템, 실제 프로덕션 환경에서의 장기 검증이 필요하다. 컴파일 비용과 시간에 대한 분석도 추가 연구 대상이다.

**종합 평가**: ⭐⭐⭐⭐ (4/5) -- 소프트웨어 공학의 TDD 철학을 LLM 에이전트에 적용한 실용적이고 참신한 접근법으로, 프로덕션 에이전트 개발의 신뢰성 문제를 체계적으로 해결할 가능성을 보여준다.

### 3. AutoAgent: Evolving Cognition and Elastic Memory Orchestration for Adaptive Agents

**저자**: Xiaoxing Wang, Ning Liao, Shikun Wei 외 | [원문](https://arxiv.org/abs/2603.09716v1) | [PDF](https://arxiv.org/pdf/2603.09716v1) | **유망점수**: 8점

# AutoAgent 논문 리뷰: 적응형 에이전트를 위한 진화하는 인지와 탄력적 메모리 오케스트레이션

## 핵심 기여
기존 자율 에이전트 프레임워크들이 **정적 인지**, **경직된 워크플로우 의존성**, **비효율적 컨텍스트 사용**이라는 문제를 가지고 있었다. AutoAgent는 이를 해결하기 위해 (1) 진화하는 인지(Evolving Cognition), (2) 실시간 맥락 기반 의사결정, (3) 탄력적 메모리 오케스트레이션이라는 세 가지 긴밀히 연결된 컴포넌트를 제안한다. 특히 외부 재훈련 없이 폐쇄 루프 인지 진화 프로세스를 통해 지속적으로 학습하는 점이 차별화된다.

## 방법론
각 에이전트는 도구, 자기 능력, 동료 전문성, 태스크 지식에 대한 **구조화된 프롬프트 수준의 인지**를 유지한다. 실행 시 이 인지와 현재 태스크 컨텍스트를 결합하여 도구 호출, LLM 생성, 에이전트 간 요청을 포함하는 통합 행동 공간에서 액션을 선택한다. **Elastic Memory Orchestrator**는 원본 기록 보존, 중복 궤적 압축, 재사용 가능한 에피소드 추상화를 통해 토큰 오버헤드를 줄이면서 결정에 중요한 정보를 유지한다.

## 실험 결과
Retrieval-augmented reasoning, 도구 활용 에이전트 벤치마크, 체화 태스크 환경에서 실험을 수행했다. 정적 베이스라인 및 메모리 증강 베이스라인 대비 **태스크 성공률, 도구 사용 효율성, 협업 견고성**에서 일관된 개선을 보였다. 특히 동적이고 비정상적(non-stationary) 환경에서 적응 능력이 두드러졌다고 보고한다.

## 한계 및 향후 연구
구체적인 정량적 수치가 초록에 제시되지 않아 실제 성능 향상 폭을 판단하기 어렵다. 메모리 압축 과정에서의 정보 손실 가능성, 복잡한 다중 에이전트 시나리오에서의 확장성 검증이 필요하다. 향후 더 복잡한 실세계 태스크, 장기간 지속 학습 시나리오에서의 검증과 인지 진화 메커니즘의 이론적 분석이 유망한 연구 방향이다.

## 🧠 Lifelong & Long-range Memory

### 4. VPWEM: Non-Markovian Visuomotor Policy with Working and Episodic Memory

**저자**: Yuheng Lei, Zhixuan Liang, Hongyuan Zhang 외 | [원문](https://arxiv.org/abs/2603.04910v1) | [PDF](https://arxiv.org/pdf/2603.04910v1) | **유망점수**: 15점

# VPWEM: 비-마르코프 시각운동 정책을 위한 작업 기억 및 에피소드 기억 리뷰

## 핵심 기여
기존 시각운동 정책들은 단일 관측이나 짧은 컨텍스트에 의존하여 장기 기억이 필요한 비-마르코프 태스크에 취약했다. VPWEM은 인간의 기억 시스템에서 영감을 받아 **작업 기억(working memory)**과 **에피소드 기억(episodic memory)**을 분리하여 설계했다. 컨텍스트 윈도우를 단순히 늘리는 대신, Transformer 기반 압축기로 과거 관측을 고정된 수의 토큰으로 압축하여 계산/메모리 비용을 거의 일정하게 유지한다.

## 방법론
최근 관측 토큰들은 슬라이딩 윈도우 방식의 **작업 기억**으로 유지하고, 윈도우를 벗어난 과거 관측은 **문맥 기억 압축기**를 통해 고정 개수의 에피소드 기억 토큰으로 재귀적으로 변환한다. 압축기는 과거 요약 토큰에 대한 self-attention과 히스토리 관측에 대한 cross-attention을 사용하며, 정책과 함께 end-to-end로 학습된다. 이를 Diffusion Policy에 적용하여 단기 및 에피소드 전체 정보를 활용한 행동 생성을 수행한다.

## 실험 결과
기억 집약적 조작 태스크 벤치마크 MIKASA에서 Diffusion Policy, VLA 모델 등 SOTA 대비 **20% 이상** 성능 향상을 달성했다. 모바일 조작 벤치마크 MoMaRT에서도 평균 **5% 개선**을 보였다. 특히 장기 기억이 필수적인 태스크에서 기존 방법들의 실패 사례를 효과적으로 해결함을 입증했다.

## 한계 및 향후 연구
논문에서 명시적 한계는 언급되지 않았으나, 압축기 학습의 안정성과 어떤 정보를 압축할지 결정하는 메커니즘의 해석 가능성이 과제로 남는다. 향후 더 긴 에피소드, 다중 태스크 환경, 그리고 실제 로봇 배포에서의 검증이 필요하다. 또한 선택적 기억 망각이나 중요도 기반 압축 전략으로 확장 가능하다.

### 5. a-TMFG: Scalable Triangulated Maximally Filtered Graphs via Approximate Nearest Neighbors

**저자**: Lionel Yelibi | [원문](https://arxiv.org/abs/2603.09564v1) | [PDF](https://arxiv.org/pdf/2603.09564v1) | **유망점수**: 5점

# a-TMFG: 근사 최근접 이웃을 통한 확장 가능한 삼각 최대 필터 그래프 논문 리뷰

## 핵심 기여
기존 TMFG는 밀집 상관행렬의 사전 계산과 저장이 필요하여 O(n²) 메모리 복잡도로 인해 대규모 데이터셋 적용이 불가능했다. 본 논문은 k-최근접 이웃 그래프(kNNG)를 초기 구조로 활용하고, 누락된 상관관계를 실시간으로 추정하는 메모리 관리 전략을 도입하여 수백만 개 관측치 규모까지 확장 가능한 a-TMFG 알고리즘을 제안한다. 이는 자연적 그래프 구조가 없는 데이터에서 지도/비지도 학습 입력용 그래프를 효율적으로 생성하는 새로운 접근법이다.

## 방법론
핵심 아이디어는 전체 상관행렬 계산을 회피하고 근사 최근접 이웃(ANN) 기반으로 초기 그래프를 구성한 뒤, TMFG의 삼각화 원리를 적용하는 것이다. kNNG로 후보 엣지를 효율적으로 탐색하고, 필요한 상관관계만 온디맨드로 계산하여 조합적 폭발을 제어한다. 이를 통해 메모리 사용량과 런타임 복잡도를 대폭 감소시키면서도 원본 TMFG의 구조적 특성을 유지한다.

## 실험 결과
수백만 개 관측치를 포함한 대규모 데이터셋에서 알고리즘의 확장성을 검증하였다. 파라미터 변화와 노이즈에 대한 강건성 테스트를 수행하여 알고리즘의 안정성을 입증했다. 기존 TMFG가 처리 불가능한 규모의 데이터에서도 실용적인 시간 내에 그래프 구성이 가능함을 보여주었다.

## 한계 및 향후 연구
근사 기반 접근법이므로 원본 TMFG 대비 정확도 손실이 존재할 수 있으며, k값 등 하이퍼파라미터 선택에 대한 가이드라인이 필요하다. ANN 알고리즘의 근사 오차가 최종 그래프 품질에 미치는 영향에 대한 이론적 분석이 부족하다. 향후 다양한 도메인(금융, 생물정보학 등) 적용 및 다운스트림 태스크 성능 평가로 확장 가능하다.

**저자**: Tingjun Dai, Mingfei Han, Tingwen Du 외 | [원문](https://arxiv.org/abs/2603.09292v1) | [PDF](https://arxiv.org/pdf/2603.09292v1) | **유망점수**: 18점

# See, Plan, Rewind (SPR) 논문 리뷰

## 핵심 기여
기존 VLA(Vision-Language-Action) 모델들은 태스크 진행 상황을 명시적으로 추적하지 않아 실패 감지 및 복구가 어려웠다. SPR은 **진행 상황 인식(progress awareness)** 개념을 도입하여, 언어 명령을 공간적 서브골 시퀀스로 분해하고 각 마일스톤 달성 여부를 모니터링함으로써 실패 시 복구 가능한 상태로 되돌아가는(Rewind) 폐루프 제어를 실현했다.

## 방법론
SPR은 세 단계 사이클로 동작한다: (1) **See** - 현재 상태와 다음 마일스톤을 시각적으로 인식, (2) **Plan** - 다음 2D 웨이포인트를 향한 궤적 계획, (3) **Rewind** - 진행이 예상 시퀀스와 어긋나면 복구 가능한 이전 상태로 회귀. 핵심은 언어 명령을 검증 가능한 중간 상태들로 동적 그라운딩하여, 추가 학습 데이터나 보조 모델 없이도 오류 수정이 가능하다는 점이다.

## 실험 결과
LIBERO 벤치마크에서 MolmoAct 베이스라인 대비 **5% 성능 향상**을 달성했다. 특히 미지의 명령과 초기 상태를 포함한 **LIBERO-Plus** 벤치마크에서 OpenVLA-OFT, UniVLA를 능가하며 **가장 적은 성능 하락폭**으로 SOTA 강건성을 입증했다. 이는 OOD(Out-of-Distribution) 환경에서의 일반화 능력이 우수함을 의미한다.

## 한계 및 향후 연구
2D 웨이포인트 기반 계획으로 인해 복잡한 3D 조작 태스크에서는 한계가 있을 수 있다. 또한 마일스톤 정의의 자동화 수준과 다양한 실제 환경에서의 검증이 추가로 필요하다. 향후 3D 공간 추론 통합, 더 정교한 실패 원인 분석, 실제 로봇 환경에서의 광범위한 평가가 유망한 연구 방향이다.

---

**종합 평가**: ⭐⭐⭐⭐ (4/5) -- 진행 상황 모니터링과 자기 복구라는 실용적 아이디어로 VLA 모델의 강건성을 의미 있게 개선한 연구이나, 실제 로봇 실험 결과가 더 풍부했다면 좋았을 것이다.

**저자**: Hongliang Zhao, Wenhui Yang, Yang Chen 외 | [원문](https://arxiv.org/abs/2603.08423v1) | [PDF](https://arxiv.org/pdf/2603.08423v1) | **유망점수**: 15점

# 논문 리뷰: AFOP-ML - 메타 학습 기반 촉각 인식

## 핵심 기여
기존 촉각 인식 연구들은 대량의 학습 데이터를 필요로 했으나, 본 논문은 **자동 특징 최적화(AFOP)**를 프로토타입 네트워크에 통합하여 few-shot 학습을 가능하게 함. 단 1개의 샘플(1-shot)만으로도 새로운 형상과 재질을 인식할 수 있어, 실제 로봇 응용에서의 데이터 수집 비용 문제를 해결. 특히 형상과 재질을 **동시에** 인식하는 통합 프레임워크를 제안한 점이 차별화됨.

## 방법론
4채널 촉각 센서 신호를 입력으로 받아 프로토타입 네트워크 기반 메타 학습 수행. 핵심은 **자동 특징 공간 최적화(AFOP)** 모듈로, 수동 특징 설계 없이 네트워크가 스스로 최적의 임베딩 공간을 학습. "Learn to learn" 패러다임을 통해 소수 샘플에서 빠르게 새로운 클래스에 적응하며, support set의 프로토타입과 query 샘플 간 거리 기반으로 분류 수행.

## 실험 결과
36개 카테고리 벤치마크에서 **5-way-1-shot 정확도 96.08%** 달성, 기존 방법들 대비 우수한 성능. 극단적인 **36-way-1-shot에서도 88.7%** 유지로 확장성 입증. 미학습 형상/재질 및 힘/속도 변화 조건에서의 일반화 실험을 통해 강건성 검증. 특히 1-shot이라는 극소 데이터 환경에서 90% 이상의 정확도는 실용적 가치가 높음.

## 한계 및 향후 연구
단일 촉각 핑거 센서에 의존하여 다중 센서 통합이나 다른 센서 모달리티와의 융합은 미검증. 정적 접촉 위주의 실험으로 동적 조작 시나리오에서의 성능은 추가 연구 필요. 향후 비전-촉각 멀티모달 학습, 실시간 로봇 조작 통합, 그리고 더 다양한 재질/환경 조건으로의 확장 가능성 열어둠.

---

**종합 평가**: ⭐⭐⭐⭐ (4/5) -- Few-shot 촉각 인식이라는 실용적 문제를 메타 학습으로 효과적으로 해결했으나, 실제 로봇 통합 검증이 보완되면 완성도가 높아질 것.
